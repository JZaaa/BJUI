样例 Class 属性
A链接按钮：A链接的按钮需要添加Class[btn]，方可转换成按钮形态，JS会为Button按钮自动添加Class[btn]。
图标说明：所用图标来自Font Awesome，使用时仅需将fa-后面的部分放入data-icon中即可。
