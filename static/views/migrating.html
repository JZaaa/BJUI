迁移说明

由于Bootstrap与表单组件存在不兼容更新，已稳定使用B-JUI 1.x的项目不建议升级
bjui next是以B-JUI框架为基础，升级修改了部分布局与组件，框架不支持IE等低版本浏览器，若有浏览器支持需要请使用B-JUI 1.x
升级说明
框架大部分组件实例化已做兼容处理
  1. 框架基于jQuery v3 与 Bootstrap 4 开发，请将项目中的jQuery与Bootstrap升级到指定版本，并进行相关迁移
  2. 更新基础框架结构
  3. 修改以下不兼容更新内容
废弃组件
以下组件已废弃，请删除组件内容
  1. 自动完成标签 Tags
  2. Datagrid
组件重构
以下组件已重构，请根据组件文档修改相关组件
  1. 侧边栏组件 sidebar，组件已重构，需要在初始化框架时传递菜单数据
  2. 日期选择器组件 datapicker，用第三方组件替换，监听回调与参数有变化
  3. 单选复选框 iCheck，原iCheck组件大量实例化下存在性能问题，现在用css实现，监听事件请用jQuery或原生事件代替
  4. 标签页 NavTab，现在不支持多标签页功能，删除所有关闭标签页操作
  5. 验证主题 nice validator theme，现在验证提示主题已重构，表单结构请参考【综合表单演示】
新增组件
新增以下组件，具体请查看相关说明文档
  1. 加载组件 Loading
  2. 消息提示拓展 Toast
