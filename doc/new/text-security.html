密码输入 该插件默认未引入框架，需手动引入

用于替换默认密码输入框，规避浏览器自动保存密码

https://github.com/noppa/text-security

引入

            <link href="BJUI/plugins/text-security/text-security-disc.min.css" rel="stylesheet">

使用

  • input标签添加classpassword-input。

    示例代码：

                        <input type="text" class="password-input">
