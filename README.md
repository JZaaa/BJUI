# BJUI-next

## 开始

### 构建

```sh
# 安装依赖
yarn install
```

```sh
# 开发模式
yarn dev
```

```sh
# 打包(不包含依赖插件)
yarn build
```

```sh
# 全部打包(包含依赖插件)
yarn build:all
```

## 演示文档

- [jzaaa.github.io/bjui-next-doc/](jzaaa.github.io/bjui-next-doc/)
- 运行`yarn dev`通过 webpack-dev-server 预览
- 自行搭建服务器，通过http服务访问`static`文件夹

## 使用

- 基础模板可复制`static`文件夹下的`index.html`进行修改
- 相关资源可复制`dist`文件夹
